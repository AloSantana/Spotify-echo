# /enhance - Enhance Existing Code Workflow

## Trigger
`/enhance <component or file>`

## Steps

1. **Analyze** current implementation with `@agent:deep-research`
2. **Profile** performance with `@agent:performance-optimizer`
3. **Review** quality with `@agent:code-reviewer`
4. **Refactor** with `@agent:jules` or `@agent:rapid-implementer`
5. **Test** improvements with `@agent:testing-stability-expert`

## Example
```
/enhance The RecommendationService.js is slow and needs Redis caching
```
