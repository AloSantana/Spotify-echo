# /create - Create New Feature Workflow

## Trigger
`/create <feature description>`

## Steps

1. **Analyze** existing code patterns in the repository
2. **Design** the feature architecture using `@agent:architect`
3. **Plan** implementation with `@agent:api-developer` for API endpoints
4. **Implement** using `@agent:rapid-implementer`
5. **Test** with `@agent:testing-stability-expert`
6. **Review** with `@agent:code-reviewer`
7. **Document** with `@agent:docs-master`

## Example
```
/create Add Spotify playlist sync feature that auto-updates playlists based on AI recommendations
```
