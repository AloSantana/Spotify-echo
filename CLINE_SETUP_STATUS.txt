Why was the original file invalid?

Cursor/VS Code use loose validation:
  ✓ Optional type field
  ✓ Direct values or placeholders for env vars
  ✓ Docker command support

Cline uses STRICT validation:
  ✓ REQUIRED type: "stdio" field
  ✓ Direct values only for env vars (NO ${input:})
  ✓ No Docker command support (use npm/npx)

The fix added:
  1. Explicit "type": "stdio" for each server
  2. Proper environment variable handling
  3. Correct command format for Cline
